## 一、基本介绍
database：数据库（强调 base）
applications：表示应用系统
DBMS：Database management system （Database Engine）
enterprise：企业，团体（强调复杂结构）
data redundancy and inconsistency：数据冗余和不一致
data isolation：数据孤岛
access data：存取数据
integrity：完整性
atomicity：原子性（不可分割）
concurrent access：并发访问

## 二、数据库的特征
data persistence：数据持久性
convenience in accessing data：数据访问便利性
data integrity：数据完整性
concurrency control for multiple users：多用户并发控制
failure recovery：故障恢复
security control：安全控制

## 三、数据库的三级抽象
view level <-> logical level <-> physical level
层级之间都有 map 映射

### 作用
1. Hide the complexities
2. Enhance the adaptions to changes
